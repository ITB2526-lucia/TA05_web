Linux Infrastructure Deployment and Network Services

This project consists of creating a complete infrastructure based on Linux servers, designed to simulate a real business environment. The main objective was to deploy essential network services, ensure their stable operation, and apply appropriate security measures.

Project Objectives

The purpose was to build a functional environment that included fundamental services such as:

  • DNS server for internal name resolution.
  • DHCP server for automatic IP address assignment.
  • FTP server for file transfer.
  • Apache web server with custom pages.
  • User and permission management in Linux.

System Architecture

The infrastructure was designed following a modular model, where each service runs on an independent virtual machine. This allows fault isolation, improved security, and easier maintenance.

Implemented Security

Several system hardening measures were applied, including:

  • Firewall configuration with specific rules.
  • Disabling unnecessary services.
  • Access control through secure SSH.
  • Permission and group management.

Technologies Used

To build this infrastructure, various tools and technologies oriented toward system administration and network services were used. These technologies allow the creation of a stable, secure, and functional environment similar to those used in real business settings.

  • Linux distributions (Debian/Ubuntu Server) for the main services.
  • DNS server (Bind9) for internal name resolution.
  • DHCP server for automatic IP address assignment.
  • FTP server for file transfer.
  • Apache web server to host content and custom pages.
  • SSH for secure remote administration.
  • UFW firewall for traffic control and system hardening.

Conclusions

This project allowed me to deeply understand how essential network services work and how they integrate within a Linux infrastructure. It also strengthened my knowledge in security, user administration, permission management, and deployment of critical services.

Thanks to this environment, I was able to simulate real scenarios, solve technical problems, optimize services, and apply best practices in system administration. It is a project that demonstrates my ability to design, configure, and maintain Linux-based infrastructures in a professional manner.
