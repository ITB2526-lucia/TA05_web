Despliegue de Infraestructura Linux y Servicios de Red

Este proyecto consiste en la creación de una infraestructura completa basada en servidores Linux, diseñada para simular un entorno empresarial real. El objetivo principal fue desplegar servicios esenciales de red, garantizar su funcionamiento estable y aplicar medidas de seguridad adecuadas.

Objetivos del proyecto

El propósito fue construir un entorno funcional que incluyera servicios fundamentales como:

  • Servidor DNS para resolución de nombres interna.
  • Servidor DHCP para asignación automática de direcciones IP.
  • Servidor FTP para transferencia de archivos.
  • Servidor web Apache con páginas personalizadas.
  • Gestión de usuarios y permisos en Linux.

Arquitectura del sistema

La infraestructura se diseñó siguiendo un modelo modular, donde cada servicio se ejecuta en una máquina virtual independiente. Esto permite aislar fallos, mejorar la seguridad y facilitar el mantenimiento.

Seguridad implementada

Se aplicaron medidas de endurecimiento del sistema, incluyendo:

  • Configuración de firewall con reglas específicas.
  • Desactivación de servicios innecesarios.
  • Control de acceso mediante SSH seguro.
  • Gestión de permisos y grupos.

Tecnologías utilizadas

Para construir esta infraestructura se utilizaron diversas herramientas y tecnologías orientadas a la administración de sistemas y servicios de red. Estas tecnologías permiten crear un entorno estable, seguro y funcional, similar al que se utiliza en entornos empresariales reales.

  • Distribuciones Linux (Debian/Ubuntu Server) para los servicios principales.
  • Servidor DNS (Bind9) para la resolución de nombres interna.
  • Servidor DHCP para la asignación automática de direcciones IP.
  • Servidor FTP para la transferencia de archivos.
  • Servidor web Apache para alojar contenido y páginas personalizadas.
  • SSH para la administración remota segura.
  • Firewall UFW para el control de tráfico y endurecimiento del sistema.

Conclusiones

Este proyecto me permitió comprender en profundidad cómo funcionan los servicios esenciales de red y cómo se integran dentro de una infraestructura Linux. También reforzó mis conocimientos en seguridad, administración de usuarios, gestión de permisos y despliegue de servicios críticos.

Gracias a este entorno pude simular escenarios reales, resolver problemas técnicos, optimizar servicios y aplicar buenas prácticas de administración de sistemas. Es un proyecto que demuestra mi capacidad para diseñar, configurar y mantener infraestructuras basadas en Linux de forma profesional.
